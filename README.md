# InvenMax IA
**Autor:** Daniel de Jesus Nogales Escalona

**Materia:** Desarrollo de Aplicaciones para Internet - Maestría en Sistemas

## Descripción del Proyecto
InvenMax IA es una plataforma inteligente diseñada para la gestión, control y optimización máxima de inventarios comerciales. El sistema combina un núcleo transaccional clásico para el registro de existencias (entradas, salidas y niveles de stock) con capacidades avanzadas de inteligencia artificial utilizando Spring AI. La plataforma analiza de forma autónoma los datos operativos en MySQL para prevenir el desabastecimiento, proyectar necesidades de compra y generar automáticamente propuestas de reabastecimiento.

## Stack Tecnológico
* **Frontend:** Angular
* **Backend:** Java + Spring Boot
* **Base de Datos:** PostgreSQL Cloud Serverless (vía Neon)
* **IA:** Spring AI + Modelos de Lenguaje
